## 카페 홈페이지 제작 (가상)
***
실제 카페 홈페이지를 만든다는 생각으로 제작하는 홈페이지

내가 카페 사장이면 어떤 홈페이지를 만들까?
***

### 페이지 구성 계획
- 홈 (모든 페이지 링크 구성)
- 어바웃(메뉴 포함)
- 물건담기(로그인 기능 구현 시,간단한 커피 굿즈)
- 매거진
- 장바구니(로그인 기능 구현 시,물건담기 탭과 연계)
- 컨택트
***

### 사용 기술
React, Tailwind CSS
Netlify 배포 예정
